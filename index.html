Conectar Google Sheets

Seguí estos pasos para sincronizar automáticamente cada evento con tu planilla.

1
Abrí Google Sheets, creá un spreadsheet nuevo y llamalo Planes Difu.
2
Andá a Extensiones → Apps Script, borrá el contenido y pegá este código:
function doPost(e) {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var data = JSON.parse(e.postData.contents);
  var sheet = ss.getSheetByName(data.hoja)
              || ss.insertSheet(data.hoja);
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(data.encabezados);
    sheet.getRange(1,1,1,data.encabezados.length)
         .setFontWeight('bold')
         .setBackground('#e8eaf6');
  }
  sheet.appendRow(data.fila);
  return ContentService
    .createTextOutput(JSON.stringify({
      ok: true, url: ss.getUrl()
    }))
    .setMimeType(ContentService.MimeType.JSON);
}
3
Hacé clic en Implementar → Nueva implementación. Tipo: Aplicación web. Acceso: Cualquier persona. Guardá y copiá la URL.
4
Pegá la URL acá abajo y guardá.
